Alumno

ERDEntity
DER Taller Marinera :: DER :: Alumno

Description

Menor de edad inscrito en el taller de marinera.

Un Alumno pertenece a un conjunto (1 o más) de SeccionesMarinera en los que practica danza y con quien participa en Eventos.
