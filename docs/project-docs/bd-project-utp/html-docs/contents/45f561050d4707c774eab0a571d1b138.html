SecciónMarinera

ERDEntity
DER Taller Marinera :: DER :: SecciónMarinera

Description

Representa un conjunto de 1 o más Alumnos a los que un cierto profesor (solo 1) les enseña marinera en cada sesión.

En una SeccionMarinera pueden estar varios alumnos inscritos al mismo tiempo.

Una SeccionMarinera puede participar en 1 o más eventos.
