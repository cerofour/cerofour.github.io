TipoMarinera

ERDEntity

Description

Representa los diferentes tipos de marinera que existen y son enseñados en el taller. Marinera limeña. Puede tomar los valores:

  • Marinera norteña.
  • Marinera puneña.
  • Marinera arequipeña.
  • Marineras andinas o serranas.
