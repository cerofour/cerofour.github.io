PiezaVestimenta

Description

Representa una pieza de una vestimenta que será alquilada para un evento. Como lo es: un pantalón, camisa, zapatos, sombrero, etc. Para un mismo Evento se pueden alquilar 1 o más piezas de vestimenta.
