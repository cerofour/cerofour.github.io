Apoderado

ERDEntity
DER Taller Marinera :: DER :: Apoderado

Description

Un apoderado es la persona encargada de manejar los procesos del alumno como pagos, inscripciones, cambio de apoderado, etc. Además, es la persona con la que se tiene contacto, por ello se le considera el número de teléfono.

Un apoderado puede estar encargado de 1 o más alumnos y con el tiempo puede cambiar el apoderado de un alumno.
