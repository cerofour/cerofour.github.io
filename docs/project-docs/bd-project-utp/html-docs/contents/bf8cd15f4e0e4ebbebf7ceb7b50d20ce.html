Evento

ERDEntity
DER Taller Marinera :: DER :: Evento

Description

Un Evento es una presentación pública de un conjunto de Secciones seleccionado.

A un Evento pueden participar 1 o más secciones. Además, en un evento generalmente se realiza un alquiler de vestimenta, donde se alquilan 1 o más PiezasVestimenta.
